# **App Name**: Kfton Experience

## Core Features:

- Landing Page & Navigation: Displays a landing page with a navigation bar containing the logo, a link to the menu, and an online shop. Upon arriving to the page, users can watch a video showcasing the essence of Kfton, accompanied by a subtle prompt guiding them downwards.
- About Us Section: Dedicated section featuring an 'About Us' overview with the mission, vision, and a photo of the founder, Santiago, as well as customer reviews and a map indicating the café's location.
- Interactive Menu: Presents an interactive menu showcasing an array of beverages, like espresso(2), americano(2,30), cortado (2,40), café con leche (2,70), cappucino(2,70), flat white(3), late(3), mocca(4), filtro batch(3,50), v60 (5) and v60 especial(8,50); teas, fresh juices, beer, natural wines, plus diverse meals.
- Optional Extras: Provides extra charge for ordering a natural pacamara coffee.
- Online Store: Enables the purchase of merchandise and coffee for home brewing, integrated with Stripe for payment processing.
- Footer Section: A footer section containing the local address, contact phone number, and a map adapted to the modern web colors.

## Style Guidelines:

- Primary color: Ash Black (#4A4A4A) evoking a modern, sophisticated feel.
- Background color: Off-White or Cream (#F8F8F2) creating a clean and inviting atmosphere.
- Accent color: Red (#FF0000) used sparingly to highlight key interactive elements like the navigation and calls to action.
- Font pairing: 'Playfair' (serif) for headlines, imparting elegance, paired with 'PT Sans' (sans-serif) for body text to ensure readability.
- Single-page design for seamless browsing with an elegant feel. The content must be structured and easy to read.
- Introduce smooth scrolling to direct the user to different parts of the page.
